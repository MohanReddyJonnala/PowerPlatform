How to Analyze Your Flow

This section provides two ways to get your flow's underlying JSON definition. The Standard Method is great for anyone, while the Advanced Method is a shortcut for admins and makers comfortable with APIs.

Standard Method: The Solution Export

1 Add your Power Automate flow into a Solution.

2 Export the entire Solution as a .zip file.

3 Unzip the file, navigate to the /Workflows/ folder, and open the flow's .json file.

4 Copy the JSON content and paste it into a GPT tool with a prompt like: ‚ÄúExplain the logic of this Power Automate flow step by step.‚Äù

5 Receive a clean, readable analysis of your flow's logic!

Advanced Method: Direct API Query

If you have admin/maker permissions, you can query the Dataverse API directly to get the flow definition without exporting anything.

Use this OData endpoint, replacing the placeholders:

https://<your-org>.crm.dynamics.com/api/data/v9.2/workflows?$filter=category eq 5 and workflowid eq <flow-id>

The clientdata field in the JSON response contains the entire flow definition you need.
